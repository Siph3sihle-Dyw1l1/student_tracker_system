Project Overview

The Student Expense Tracker System is a desktop application developed in Java using the Swing library, primarily built within the Apache NetBeans IDE. This project was developed as a core academic assignment at Tshwane University of Technology (TUT) to demonstrate proficiency in Object-Oriented Programming (OOP) principles, data input validation, calculation logic, and file persistence.

The application allows students to input their monthly allowance and track up to three key expenses, providing a clear summary and calculating the remaining balance.

Features

GUI Interface: User-friendly graphical interface built with Java Swing for simple data entry.

Expense Tracking: Allows users to name and assign monetary values to three major monthly expenses.

Budget Validation: Automatically checks if the total expenses exceed the monthly budget upon saving.

Financial Summary: Displays a detailed summary showing the monthly budget, individual expenses, total spent, and the current balance.

Data Persistence: Saves and retrieves student expense records to/from a local text file (ExpenseList.txt) using Java File I/O operations.

History Display: Can display the full transaction history of all students in a structured format within the application's text area.

Technologies Used

Core Language: Java

GUI Framework: Java Swing

Development Environment: Apache NetBeans

Data Persistence: Java File I/O (FileReader, FileWriter, BufferedReader, BufferedWriter)
